Acceptable HotKeys:
CTRLLEFT, CTRLRIGHT, CTRLUP, CTRLDOWN
F Buttons F1 to F9
CTRLF1 To CTRLF9
ALTF1 to ALTF9
CTRL+ any single key CTRLA, CTRL1 etc
ALT+ any single key ALTA, ALT1 etc

Instructions:
In OpenScrape highlight symbols in the left window and click New.
Select a record name like i0buttonhotkey.
Then enter the String for your hotkey.

If your casino isn't covered let me know.
Input for the table map string is not case sensitive.
